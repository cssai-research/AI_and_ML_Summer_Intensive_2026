# USF AI and ML Summer Intensive 2026

The **USF AI and ML Summer Intensive 2026** was a five-day camp for K–12 students, offered jointly by **USF Youth Experiences** and the **USF Bellini College of Artificial Intelligence, Cybersecurity, and Computing**.

The camp introduced students to foundational concepts in Python programming, machine learning, deep learning, and large language models through lectures, hands-on activities, and a collaborative group project.

## Camp Schedule

### Day 1: Introduction to Python Programming

Students learned the fundamentals of Python programming, including variables, data types, conditional statements, loops, functions, and basic data manipulation.

### Day 2: Machine Learning and Guest Lecture

Students were introduced to core machine learning concepts and participated in hands-on activities involving data, features, model training, prediction, and evaluation.

The day also included a guest lecture.

### Day 3: Deep Learning and Group Project Launch

Students explored the basic ideas behind neural networks and deep learning through interactive examples and practical exercises.

Students also formed groups and began working on their final projects.

### Day 4: Large Language Models, Group Project, and Guest Lecture

Students learned how large language models work and explored how generative AI applications can be built using prompts and application programming interfaces.

Groups continued developing their projects, and the day included an additional guest lecture.

### Day 5: Project Wrap-Up and Final Presentations

Students completed their group projects and presented their work to instructors, peers, and camp participants.

## Learning Goals

By the end of the camp, students had opportunities to:

* Write and run introductory Python programs
* Work with data using computational tools
* Understand the basic machine learning workflow
* Explore neural networks and deep learning
* Learn how large language models can be used in applications
* Collaborate on an AI or machine learning project
* Communicate technical ideas through a final presentation
